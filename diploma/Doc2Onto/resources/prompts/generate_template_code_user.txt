Сгенерируй Python-код шаблона извлечения.

### Входные данные

#### 1. Описание шаблона
$template_description

#### 2. Пример документа в формате UDDM
$uddm_example

#### 3. Код шаблона, который тебе надо взять за основу и заполнить
$code_example

### Требования
1. Реализуй класс TemplateCode
2. Заполни:
  - classify
  - fields
  - build_triples
3. Правила:
  1. CLASSIFY:
    - Используй doc_name или характерные текстовые якоря
    - Не делай сложную логику
  2. FIELDS:
    - Каждый Field должен соответствовать одному полю из описания
    - name — snake_case
    - description — склеенное описание (столбцы: описание + где искать + как извлекать)
    - selector — строго по "Где искать"
    - extractor — строго по "Как извлекать"
    - validator:
      - required для обязательных полей
      - regex / range если применимо
  3. BUILD_TRIPLES:
    - Используй fields_values
    - Формируй триплеты в виде:
      {"subject": ..., "predicate": ..., "object": ...}
    - Используй осмысленные URI/предикаты
    - Не усложняй

### Ответ

Формат ответа:
- Полный текст кода (включая импорты)
- Без пояснений
- Без markdown
- Без ``` блоков

Перед выводом кода проверь:
- все поля из описания присутствуют
- нет несуществующих методов
- код компилируется логически
