Тебе дан документ, список полей, которые не удалось извлечь автоматически, и описание шаблона извлечения (который использовался для декларативного извлечения полей).

Твоя задача — попытаться извлечь значения этих полей из текста документа.

### Входные данные

#### 1. Документ в формате UDDM
$document_uddm_text

#### 2. Описание шаблона
$template_description

#### 3. Поля для извлечения
$fields

### Требования
- Используй описание поля как подсказку (что именно искать)
- Учитывай возможные вариации формулировок
- Если значение не найдено — верни null
- Не извлекай лишний текст (только значение)

### Формат ответа

Для каждого поля укажи:
- значение, которое ты извлёк (если получилось)
- null (в противном случае)

Формат ответа — строго JSON вида:
{
  "field_name": "value" | null,
  ...
}
