Тебе дан текст документа и результаты извлечения полей.

Твоя задача — провалидировать и при необходимости исправить значения полей.

---

Текст документа:
$document_text

Поля:
$fields

---

Для каждого поля:
1. Проверь соответствие значения описанию
2. Проверь наличие подтверждения в тексте документа
3. При необходимости исправь значение
4. Если восстановление невозможно — укажи причину

Формат ответа:
{
  "field_name": {
    "status": "valid | corrected | invalid",
    "value": "строка или null",
    "error": "null или сообщение об ошибке"
  }
}

---

Требования:

- Если поле корректно → status = "valid", value без изменений
- Если исправлено → status = "corrected", value = исправленное значение
- Если невозможно восстановить → status = "invalid", value = null, error обязателен

- Ошибки формулируй кратко и по делу
- Не дублируй входной error, если он неактуален
- Не добавляй новые поля
- Не пропускай поля
