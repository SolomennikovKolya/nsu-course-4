Тебе дан payload с информацией по полям.

Твоя задача — провалидировать и при необходимости исправить значения полей.

### Входные данные

#### Результаты извлечения и предварительной валидации (payload)
$fields

### Требования

Для каждого поля:
1. Проверь соответствие значения описанию
2. При необходимости исправь значение
3. Если восстановление невозможно — укажи причину

### Формат ответа

Формат ответа — строго JSON.
Никакого дополнительного текста.

Формат JSON-а:
{
  "field_name": {
    "status": true | false,        - итоговый статус корректности
    "value": "строка или null",    - исправленное значение или null (если значение и так валидно или его невозможно исправить)
    "error": "null или сообщение"  - ошибка/предупреждение
  }
}

Возможные случаи:
- Если поле корректно и значение не требовало правки → status = true, value = null, error = null
- Если поле исправлено → status = true, value = исправленное значение, error = краткое предупреждение (что было исправлено)
- Если невозможно восстановить → status = false, value = null, error = сообщение ошибки (кратко и по делу)
