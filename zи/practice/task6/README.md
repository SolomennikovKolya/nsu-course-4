
## Firewall

[Подробное задание](https://disk.yandex.ru/i/39Xr5XGmT30NeA)

### Задание

Цели:
- Ознакомиться с возможностями межсетевого экрана операционной системы Windows
- Изучить последовательность операций по включению и настройке межсетевого экрана
- Приобрести практические навыки по защите компьютера с помощью механизма межсетевого
экранирования

Задание - активизировать встроенный брандмауэр операционной системы Windows и настроить его параметры:
1. Запретить ping ресурсов интернет
2. Запретить доступ в интернет с одного из браузеров, установленных на ПК (Пример. Из Chrome доступ в интернет заблокирован, а IE все работает)
3. Запретить доступ пользователей к ресурсам например mail.ru

Предоставить отчет, содержащий: 
- Скриншоты правил «Брандмауэр в Windows». На которых видно параметры блокировки
- Скриншоты выполнения команды ping в cmd на которой видно, что ping ресурса заблокирован и блокировки браузера (как в примере)
