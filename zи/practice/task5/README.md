
## Статический анализатор кода

[Папка со всеми материалами](https://disk.yandex.ru/d/S0BNE3JRyEjcYA)

### Задание

В рамках лабораторной будет изучен статический анализатор кода AppCheker, предназначенный для автоматизированного поиска дефектов в исходном коде приложений, разработанных на С#, C/C++, Java, PHP

Задачи:
1. Установить сканер безопасности на ПК. Это может быть, как Ваш личный ПК. Также установщик есть для unix-систем
2. Войти в анализатор года (перед запуском смените время на вашем ПК на 2022 год)
3. Создать проект по анализу тестового образца. Также вы можете загрузить любой свой образец на C/C++
4. Произвести сканирование образца

Предоставить отчет, содержащий: 
- Описание одного из анализатора кода, который представлены на отечественном рынке, с его плюсы по сравнению с другими
- Скриншоты отчета о найденных дефектах в тестовом образце по результатам
сканирования AppCheker
