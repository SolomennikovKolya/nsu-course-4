
## OSINT

### Задание

В рамках лабораторной будет изучен функционал DLP-системы Falcongaze предназначенной для контроля документооборота, которая позволяет вовремя предупредить утечку конфиденциальной информации. Под наблюдением системы находятся все информационные каналы, от электронной почты до мессенджеров и USB-устройств.

Задачи на лабораторную работу:
1.	Установить демоверсию DLP-системы Falcongaze (файл - fgstsetup_6.6_nfr_ru.zip)
2.	После установки инсталлятор вам предложит импортировать демобазу.
3.	После импорта демобазы зайдите в консоль пользователя.
4.	Выполнить задание, описанное в файле «Написание правил и политик для DLP-системы.docx».

В результате от Вас жду отчет, содержащий: «отчет о суммарной активности одного из пользователей» и пример конфигурации правила безопасности при помощи которого можно отследить передачу файлов через электронную почту или меседжер.

Ссылка на дистрибутив - https://disk.yandex.ru/d/wUidgmSOScg0AA
